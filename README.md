# division

Use `division` to divide two numbers in an easy way.

## Example

```ts
import { division } from '@flacial/division'

console.log(division(10, 2)) // => 5
```

## Install

```sh
deno add @flacial/division
```

## API

### division(a: number, b: number): number

Divide two numbers.

#### a

Type: `number`

The first number.

#### b

Type: `number`

The second number.
